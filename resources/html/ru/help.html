Интерфейс пользователя

Tab  –  показать подсказки автодополнения.

Enter  –  применить изменения и сбросить фокус.

Стрелки Влево и Вправо –  показать предыдущее/следующее изображение.

Настройки

%1  –  замена тегов некоторых имиджборд на их более короткие версии.

%2  –  восстановление оригинальных тегов имиджборд.

Прокси

%6 использует сеть только когда выполняется Реверс-поиск изображения при помощи сервиса iqdb.org. В некоторых случаях прокси нужна для доступа к сети, либо для защиты Вашей приватности (к примеру, с помощью Tor).

Указать URL прокси можно с помощью опции командной строки: 

--proxy=socks://localhost:9050
Эта опция не сохраняется, для этого нужно указывать её каждый раз. Это удобнее всего сделать модифицировав ярлык на программу.

Сейчас прокси %3%4.

Портативный режим

%6 поддерживает работу в портативном режиме. Чтобы его включить, создайте файл portable.dat в папке программы.

Когда приложение работает в портативном режими, все настройки сохраняются в .ini-файле в папке программы, а системный реестр не используется.

Портативный режим сейчас %5.
