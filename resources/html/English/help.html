User Interface

Tab  –  list autocomplete suggestions.

Enter  –  apply changes and clear focus.

Left and Right arrows  –  show previous/next picture.

Settings

%1  –  replaces certain imageboard tags with their shorter version.

%2  –  restores replaced imageboard tags back to their original version.

Proxy

%4 accesses internet only when Reverse Searching a picture using iqdb.org service. In some cases a proxy is needed to access internet, or to protect your privacy (using Tor for example).

Portable Mode

%4 supports running in portable mode. To enable it, create file portable.dat inside application's directory.

When portable mode is enabled, all settings will be saved in an .ini file inside application's directory and system registry will not be used.

Portable mode is currently %3.

More documentation at project repository page.
